# Cycle 1 Agent Contract: Passive Profiling & Local Intake

This contract defines the operational parameters, templates, and execution guidelines for Cycle 1 research sprints.

---

## 🎯 Cycle 1 Targets
1. **Debate Scope**: Sprints consisted of exactly **8 topics** per research run (profiling 64 total entities).
2. **Evaluation Coordinates**: Matrix positioning was manually assigned to strategic quadrants (Maturity vs. Autonomy) without dynamic composite scaling.
3. **Audit Constraints**: Static markdown profiles published with basic layout guidelines, without self-correcting Critic audits.

---

## 🔬 Core Protocols
* **Routing Standard**: Direct manual script execution rather than unified routing via an orchestrator entry point.
* **Context Budget**: Full workspace loading without progressive disclosure context budgeting, resulting in high context window pressure.
* **Local Recovery**: Manual intervention required for environmental or host failures; no automated diagnostic scripts or OOM recovery enclaves.
